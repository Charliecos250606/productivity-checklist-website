# Productivity Self-Assessment Website

This is a frontend-only productivity self-assessment website that presents users with 60 questions to help evaluate their productivity habits.

The project is mainly focused on the user interface and layout. It does not currently include backend functionality or a complete scoring system, and user responses are not stored.

### Technologies Used
- HTML
- CSS
- JavaScript
